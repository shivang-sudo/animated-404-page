# 🏔️ Animated 404 Error Page

<p align="center">

An interactive **404 Error Page** built using **HTML, CSS, JavaScript, SVG, and GSAP**.

Smooth animations, dynamic lighting effects, and an animated Yeti character create a modern **Page Not Found** experience.

</p>

in your browser.
